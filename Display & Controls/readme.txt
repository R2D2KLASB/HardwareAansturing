KEYWORDS

Om het display aan te sturen moet er via de native poort serieel gestuurd worden:

1: Boot van player verwijderen.

2: Hit bij player toevoegen.

3: Miss bij player toevoegen.

4: Boot van enemy verwijderen.

5: Hit bij enemy toevoegen.

6: Miss bij enemy toevoegen.

7: Om de text aan te passen die op het scherm staat. Gelijk na de 6 de string sturen die op het scherm moet.
	Voorbeeld '6FIRE'.
8: reset alle data op het scherm
